# OodleSharp
OodleSharp is a .NET library that can Compress and Decompress Oodle files.

It is a wrapper that functions on top of a 64-bit Oodle 2.3.0 DLL file.

## Supported Formats:
* LZH
* LZHLW
* LZNIB
* None
* LZB16
* LZBLW
* LZA
* LZNA
* Kraken
* Mermaid
* BitKnit
* Selkie
* Akkorokamui

## Supported Compression Levels:
* None
* SuperFast
* VeryFast
* Fast
* Normal
* Optimal1
* Optimal2
* Optimal3
* Optimal4
* Optimal5
